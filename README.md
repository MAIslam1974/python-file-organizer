# python-file-organizer 
A simple Python script to automatically organize files in a directory by file type and modification date.

## Features
- Sorts files into folders like Images, Documents, Videos, etc.
- Works on Windows, macOS, and Linux

## Technologies
- Python 3
- `os`, `shutil` libraries

## How to Use
1. Place the script in a folder with unsorted files
2. Run the script using `python file_organizer.py`
3. Files will be moved into categorized folders
